# License

This repository should use a split-license model.

## Training content

Recommended license for curriculum, slides, handouts, checklists, and documentation:

**Creative Commons Attribution 4.0 International — CC BY 4.0**

This allows reuse and adaptation with attribution.

## Code and lab infrastructure

Recommended license for original code, scripts, Docker files, and lab infrastructure:

**Apache License 2.0** or **MIT License**

## Third-party labs

Third-party labs such as DVAIA keep their own licenses.

Do not copy third-party lab content into this repository unless the license allows it. Prefer linking to the original project and writing independent lab wrappers, notes, and mappings.
