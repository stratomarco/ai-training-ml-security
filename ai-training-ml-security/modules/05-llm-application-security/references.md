## Suggested reading focus

For this module, students should focus on:

1. Prompt injection as a system design problem.
2. Insecure output handling as classic injection through generated content.
3. Sensitive disclosure as an access-control and data-minimization failure.
4. Excessive agency as a workflow and privilege problem.
5. Overreliance as a human/system trust problem.
