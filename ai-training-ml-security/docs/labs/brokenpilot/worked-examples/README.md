# BrokenPilot Worked Examples

These examples help instructors calibrate grading and help students understand the expected quality of capstone deliverables.

The examples are intentionally short. They are not full solutions. They illustrate the difference between vague and actionable security work.

## Included examples

- Strong threat model example
- Weak threat model example
- Strong risk register example
- Weak risk register example
- Strong executive readout example
- Weak executive readout example

## How to use

Instructors can use these examples during capstone preparation or grading calibration. Students can review them before final submission to understand the expected level of specificity.
