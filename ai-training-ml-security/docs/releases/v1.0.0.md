# AI Training — ML Security v1.0.0

## Summary

This is the first complete teachable release of **AI Training — ML Security**.

The course teaches security engineering for systems that contain machine learning, including classical ML, LLM applications, RAG, AI agents, secure MLOps, privacy attacks, adversarial ML, AI red teaming, and a final BrokenPilot capstone.

## Included

- Twelve complete modules.
- BrokenPilot capstone.
- Lab wrappers.
- Instructor notes.
- Student handouts.
- Markdown slide decks.
- Exercises.
- Quizzes and answer keys.
- Checklists.
- Templates.
- Assessments.
- References.

## Capstone

The BrokenPilot capstone is a realistic internal AI agent scenario used to assess whether students can:

- Understand an AI architecture.
- Identify trust boundaries.
- Threat model the system.
- Find and explain vulnerabilities.
- Build attack narratives.
- Design mitigations.
- Communicate residual risk.
