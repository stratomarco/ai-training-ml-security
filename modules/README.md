# Modules Overview

The course contains twelve modules. Modules 05, 06, 07, and 10 include expanded reading-first deep dives, attack anatomy, controls and remediations, common mistakes, and worked examples.

| Module | Focus |
|---|---|
| 01 | Security engineering foundations for AI |
| 02 | ML system architecture |
| 03 | OWASP ML Security Top 10 |
| 04 | BIML architectural risk analysis |
| 05 | LLM application security |
| 06 | RAG security and indirect prompt injection |
| 07 | Agent and tool security |
| 08 | Secure MLOps and AI supply chain |
| 09 | Privacy attacks and data protection |
| 10 | Adversarial ML and robustness |
| 11 | AI red team methodology |
| 12 | BrokenPilot capstone |
