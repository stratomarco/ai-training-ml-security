# Release Note: 40-Hour Delivery Profiles

Added a realistic 40-hour delivery profile layer for the one-week professional training model.

## Added

- Per-module `delivery-profile.md` files for Modules 01 through 12.
- Instructor guide: `instructor/40-hour-module-delivery-profiles.md`.
- Student-facing course map: `docs/start-here/40-hour-course-map.md`.
- Assessment map: `assessments/40-hour-deliverables-map.md`.
- Validation script: `scripts/check_delivery_profiles.py`.
- MkDocs navigation patch script: `scripts/add_delivery_profile_nav.py`.

## Purpose

This update makes the 40-hour course practical by separating live core content from compressed coverage, lab work, and self-study.
