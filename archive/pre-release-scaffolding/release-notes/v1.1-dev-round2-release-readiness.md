# v1.1-dev round 2 release-readiness fixes

This update addresses the second external review of the v1.1-dev training package.

## Closed items

- Completed site navigation coverage for high-value student material.
- Added explicit navigation for course templates, labs, delivery profiles, assessments, instructor material, and release notes.
- Added explicit `not_in_nav` handling for intentional non-navigation files.
- Added a strict MkDocs build path so future orphaned pages and broken links fail CI.
- Added a GitHub Actions quality workflow that runs the generated MkDocs site build and BrokenPilot pytest.
- Added writing style guidance that treats punctuation as a style preference, not a release gate.
