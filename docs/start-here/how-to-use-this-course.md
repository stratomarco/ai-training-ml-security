# How to Use This Course

Each module follows the same structure:

1. Read the module overview.
2. Review the slides.
3. Read the student handout.
4. Complete the exercise.
5. Run the lab or tabletop scenario, when applicable.
6. Use the checklist and templates to produce a deliverable.
7. Take the quiz.
8. Review the instructor notes if you are teaching or self-studying.

## Learning mode

For self-study, complete one module at a time and write the deliverable for each module.

For instructor-led delivery, use the slides and instructor notes as the teaching backbone, then run the exercises as group work.

## Deliverables matter

The course is designed around practical security artifacts, not only attack demonstrations. Students should produce threat models, risk registers, red team reports, architecture reviews, and residual risk statements.
