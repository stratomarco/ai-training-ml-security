# Prerequisites

Students should ideally understand:

- Basic web application architecture
- APIs
- Authentication and authorization
- Common web vulnerabilities
- Basic cloud concepts
- Basic Python or scripting
- Basic ML terms: dataset, model, training, inference, evaluation

Students do not need to be ML researchers. The course explains ML concepts only as deeply as needed for security reasoning.

## Local tools

Some labs are tabletop exercises and require only the course material. Other labs may require an external vulnerable AI application, such as DVAIA, or a small local ML environment.

Recommended local tools:

- Git
- Python 3.10+
- Docker Desktop, if running vulnerable lab environments
- A code editor such as VS Code
- A browser
