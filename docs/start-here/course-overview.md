# Course Overview

AI Training  -  ML Security is a practical curriculum for security engineering in systems that contain machine learning.

The course connects classic security foundations with ML, LLM, RAG, agent, MLOps, privacy, adversarial ML, and red team topics.

## What students learn

Students learn to:

- Threat model AI-enabled systems.
- Identify ML, LLM, RAG, and agent attack surfaces.
- Use OWASP, BIML, NIST, MITRE ATLAS, and classic security engineering literature.
- Perform labs using controlled vulnerable environments.
- Design mitigations that balance security, usability, and developer velocity.
- Produce engineering deliverables such as threat models, architecture reviews, risk registers, red team reports, and residual risk statements.

## Recommended path

1. Read the course map.
2. Complete Modules 01–04 for foundations.
3. Complete Modules 05–07 for LLM, RAG, and agent security.
4. Complete Modules 08–11 for MLOps, privacy, robustness, and red teaming.
5. Complete Module 12 as the capstone.
