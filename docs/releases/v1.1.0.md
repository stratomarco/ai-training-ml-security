# AI Training ML Security v1.1.0  -  Website Edition

This release adds a MkDocs Material website layer to the curriculum.

## Added

- MkDocs Material configuration in `mkdocs.yml`
- Python dependencies in `requirements.txt`
- GitHub Pages deployment workflow in `.github/workflows/deploy-docs.yml`
- Website landing page at `docs/index.md`
- Start Here section
- Lab Setup section
- DVAIA prerequisite page
- Local preview instructions
- GitHub Pages publishing instructions
- Mirrored module, lab, template, instructor, and assessment content under `docs/`

## What changed

The curriculum content remains the same. The major change is usability: students can now browse the course as a website instead of navigating the GitHub file tree manually.

## Deployment

Preview locally:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
mkdocs serve
```

Publish with GitHub Pages by enabling **Settings → Pages → Source: GitHub Actions** and pushing to `main`.

## Notes

For v1.1, the course content is mirrored into `docs/` for publishing while root-level folders are preserved for compatibility with the v1.0 repository layout.
