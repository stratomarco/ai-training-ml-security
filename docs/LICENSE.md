# License

This repository uses a split-license model.

## Course content

Unless otherwise stated, Markdown curriculum content, slides, instructor notes, student handouts, templates, checklists, assessments, and documentation are intended to be licensed under:

**Creative Commons Attribution 4.0 International — CC BY 4.0**

This means others may share and adapt the training material, including for commercial purposes, as long as they provide appropriate attribution.

## Future code and lab infrastructure

Original code, scripts, Docker files, test applications, and lab infrastructure added later should use a software license such as:

**Apache License 2.0**

or

**MIT License**

The final choice should be made before adding substantial code.

## Third-party labs and references

Third-party projects, including DVAIA and similar vulnerable AI/LLM training environments, keep their own licenses.

This repository should link to those projects and provide original lab wrappers, guidance, mappings, and teaching notes. Do not copy third-party lab content into this repository unless the license permits it.

## Before public release

Before making the repository public, review whether this split-license policy is sufficient for your use case. If needed, add formal license files such as:

- `LICENSE-CONTENT-CC-BY-4.0.md`
- `LICENSE-CODE-APACHE-2.0.md`

This file is a practical project license policy, not legal advice.
