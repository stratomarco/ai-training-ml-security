# v1.1-dev Lab Improvement Part 3: Toy Classifier

This package adds a separate, deterministic toy-classifier target for Modules 03 and 10.

## Why

BrokenPilot is the right target for LLM, RAG, agent, privacy, and red-team application flows. It is not the right target for classical ML model behavior. This package follows the one-target-per-modality principle by adding a small classifier app for evasion, poisoning, extraction, and output-integrity demonstrations.
